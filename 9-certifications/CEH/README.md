# CEH (Certified Ethical Hacker)

CEH, de **EC-Council**, es una certificación **amplia y teórica** que recorre las
fases del hacking ético, herramientas y conceptos. Es reconocida en el mercado,
especialmente para perfiles que empiezan y roles corporativos/gubernamentales.

> ⚠️ El material oficial de CEH tiene **copyright de EC-Council**. Aquí **no se
> redistribuye**: estudia desde la fuente oficial y con laboratorios legales.

## 🛠️ Cómo prepararse (recursos legales)
- [Página oficial CEH](https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/)
- Práctica: [TryHackMe](https://tryhackme.com/paths) · [HTB Academy](https://academy.hackthebox.com/)

## 🧪 Ejercicios básicos (individuales)
1. Enumera las fases del hacking ético que evalúa CEH.
2. Relaciona cada fase con un módulo de este curso.
3. Crea un plan de estudio de 8 semanas con hitos.

## 👥 Tarea colaborativa en equipo
El grupo debe, en una rama:
1. Construir un banco de preguntas de repaso **propias** (no copiadas del oficial).
2. Compartir recursos gratuitos de estudio.
3. Enviar PR (ver [`CONTRIBUTING.md`](../../CONTRIBUTING.md)).
