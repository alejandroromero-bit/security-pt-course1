# OWASP

OWASP (Open Worldwide Application Security Project) es una fundación sin ánimo de
lucro que produce **estándares, guías y herramientas abiertas** de seguridad web.
Su recurso más famoso es el **OWASP Top 10**, la lista de las categorías de riesgo
más críticas en aplicaciones web.

## 📁 Material hospedado
- [`OWASP_Top_10.pdf`](OWASP_Top_10.pdf) — documento oficial (licencia abierta).

## 🛠️ Recursos/proyectos OWASP clave
[Top 10](https://owasp.org/www-project-top-ten/) · [WSTG](https://owasp.org/www-project-web-security-testing-guide/)
· [ASVS](https://owasp.org/www-project-application-security-verification-standard/)
· [Juice Shop](https://owasp.org/www-project-juice-shop/) · [ZAP](https://www.zaproxy.org/)

## 🧪 Ejercicios básicos (individuales)
1. Lee el `OWASP_Top_10.pdf` y resume cada categoría en una frase.
2. Relaciona 3 categorías con vulnerabilidades reales que conozcas.
3. Explora un proyecto OWASP (ZAP o Juice Shop) y describe para qué sirve.
