# 🧪 Ejercicios — Seguridad de aplicaciones web

> ⚠️ Todo se practica en [PortSwigger Academy](https://portswigger.net/web-security)
> (labs gratuitos y legales) o [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/).

## Ejercicio 1 · Mapear el OWASP Top 10
**Objetivo:** interiorizar las categorías.
1. Lee el [`OWASP_Top_10.pdf`](OWASP/OWASP_Top_10.pdf) hospedado.
2. Crea una tabla: categoría → ejemplo real → mitigación básica.

## Ejercicio 2 · Tu primera inyección SQL (lab)
**Objetivo:** entender SQLi en un entorno controlado.
1. Completa el lab introductorio de SQL injection en PortSwigger Academy.
2. Explica por qué la consulta es vulnerable y cómo lo evitarían consultas
   parametrizadas.

## Ejercicio 3 · XSS reflejado (lab)
**Objetivo:** ejecutar un XSS de práctica.
1. Resuelve el lab "Reflected XSS into HTML context" de PortSwigger.
2. Describe el impacto real si ocurriera en una app en producción.

## 🏁 Reto
Recorre OWASP Juice Shop y encuentra **una vulnerabilidad de cada** de estas
clases: control de acceso roto, inyección y XSS. Documenta evidencia y mitigación
para cada una.
