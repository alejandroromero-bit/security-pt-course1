# Wireshark

Wireshark es el **analizador de protocolos** de referencia: captura y disecciona
el tráfico de red paquete a paquete. Es tan útil en ofensiva (ver credenciales en
claro, entender protocolos) como en defensa/forense (analizar un incidente a
partir de un `.pcap`).

## 🛠️ Herramientas habituales
`Wireshark` (GUI) · `tshark` (CLI) · `tcpdump` · filtros de captura (BPF) y de
visualización (display filters)

## 🧪 Ejercicios básicos (individuales)
> Captura **solo** tu propio tráfico.
1. Captura mientras navegas y aplica el filtro `http`.
2. Usa "Follow → HTTP Stream" para reconstruir una petición completa.
3. Filtra por `dns` y localiza consulta y respuesta.
