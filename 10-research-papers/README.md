# 10 · Papers e investigación (Research & CVE analysis)

Un buen profesional se mantiene al día leyendo investigación y analizando
**vulnerabilidades reales (CVEs)**. Este módulo enseña a encontrar, leer e
interpretar avisos de seguridad y a evaluar su impacto.

## 🎯 Objetivos de aprendizaje
- Buscar y leer CVEs en fuentes oficiales.
- Interpretar el vector y la puntuación **CVSS**.
- Priorizar según explotación activa (CISA KEV).
- Leer un aviso técnico y extraer causa raíz y mitigación.

## 🧠 Conceptos clave
- **CVE:** identificador único de una vulnerabilidad pública.
- **NVD:** base de datos del NIST que enriquece cada CVE (CVSS, CWE, referencias).
- **CWE:** clasificación del tipo de debilidad (p. ej. CWE-79 = XSS).
- **CISA KEV:** catálogo de vulnerabilidades **explotadas activamente** — prioridad máxima.
- **Causa raíz:** el fallo de fondo, no solo el síntoma.
