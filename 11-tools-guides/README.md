# 11 · Herramientas y guías (Tools & Guides)

Un pentester es tan bueno como su dominio de las herramientas base. Este módulo
recopila utilidades esenciales y guías de referencia para **Linux** y **Windows**,
incluyendo la "navaja suiza" de las redes: **Netcat**.

## 🎯 Objetivos de aprendizaje
- Dominar utilidades de línea de comandos imprescindibles.
- Usar Netcat para transferencias, escaneos simples y shells (en laboratorio).
- Conocer binarios abusables en Linux (GTFOBins) y Windows (LOLBAS).
- Moverte con soltura por el sistema de archivos y los permisos.

## 🧠 Conceptos clave
- **Netcat:** leer/escribir en conexiones TCP/UDP; útil para listeners y pruebas.
- **LOLBins / GTFOBins:** binarios legítimos del sistema abusados para tareas
  ofensivas (persistencia, escalada, exfiltración).
- **Living off the land:** operar con herramientas ya presentes para evitar detección.
- **Permisos y SUID:** base de muchas escaladas de privilegios en Linux.

## 🛠️ Herramientas típicas
`netcat`/`ncat` · `curl`/`wget` · `ssh`/`scp` · `find`/`grep`/`awk` · PowerShell
