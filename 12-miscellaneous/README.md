# 12 · Misceláneo (Frameworks y gobernanza)

La ciberseguridad no es solo ofensiva: los **frameworks** dan estructura, lenguaje
común y criterios de madurez. Este módulo reúne referencias de gobernanza y buenas
prácticas de **libre distribución**.

## 🎯 Objetivos de aprendizaje
- Conocer los frameworks de referencia (NIST CSF, CIS Controls, NCSC CAF).
- Entender para qué sirve un framework y cómo se aplica.
- Situar el pentesting dentro de un programa de seguridad más amplio.

## 🧠 Conceptos clave
- **NIST CSF:** funciones Identificar, Proteger, Detectar, Responder, Recuperar
  (y en la 2.0, Gobernar).
- **CIS Controls:** conjunto priorizado de controles defensivos prácticos.
- **NCSC CAF:** marco de evaluación de ciberseguridad del Reino Unido.
- **Gobernanza:** políticas, roles y métricas que sostienen la seguridad.
