# OSINT (Open Source Intelligence)

OSINT es el arte de obtener **inteligencia a partir de fuentes públicas**: webs,
redes sociales, registros WHOIS, certificados, filtraciones y metadatos. Es la
base del reconocimiento **pasivo**: se puede recopilar muchísima información de un
objetivo **sin enviarle un solo paquete**, por lo que es casi indetectable.

Un atacante usa OSINT para mapear personas, tecnologías y correos; un defensor lo
usa para saber **qué expone su organización** antes de que lo haga alguien más.

## 🛠️ Herramientas habituales
`theHarvester` · `Recon-ng` · `SpiderFoot` · `Maltego` · `Sherlock` · `Shodan`
· Google Dorks · [OSINT Framework](https://osintframework.com/)

## 🧪 Ejercicios básicos (individuales)
1. Con `theHarvester`, recopila correos y subdominios de un dominio de práctica.
2. Usa 3 *Google dorks* (`site:`, `filetype:`, `intitle:`) y explica qué revela cada uno.
3. Investiga tu propia huella pública y anota qué reducirías por privacidad.

## 👥 Tarea colaborativa en equipo
Este es un **punto de trabajo en equipo**. El grupo asignado a OSINT debe:
1. Crear una rama (`git checkout -b osint-equipo`).
2. Investigar y **completar la sección de abajo** con herramientas, comandos y
   trucos que hayan probado (¡solo en objetivos autorizados!).
3. Abrir un Pull Request para que el resto lo revise. Ver [`CONTRIBUTING.md`](../../CONTRIBUTING.md).
