# Forense digital (Digital Forensics)

La forense digital recupera, preserva y analiza **evidencia** de dispositivos:
memoria RAM, discos y artefactos del sistema. El objetivo es reconstruir qué pasó
respetando la **cadena de custodia** para que la evidencia sea válida.

## 🛠️ Herramientas habituales
`Autopsy` / `The Sleuth Kit` · `Volatility` (memoria) · `FTK Imager` · `foremost`
(file carving) · hashing para integridad (`sha256sum`)

## 🧪 Ejercicios básicos (individuales)
> Usa muestras públicas de práctica, nunca datos reales de terceros.
1. Investiga el **orden de volatilidad** (qué se recolecta primero y por qué).
2. Con `Volatility` sobre un volcado de práctica, lista procesos.
3. Explica qué es *file carving* y cuándo se usa.
